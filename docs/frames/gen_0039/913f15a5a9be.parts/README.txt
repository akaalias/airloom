Airloom evolved frame -- printable/cuttable templates

Material: PA12-CF (carbon-fiber nylon)
Arm thickness: 4.3 mm (the STLs are already extruded to thickness)
Deck plate thickness: 1.6 mm

Parts to produce:
  2x front arm (arm_front.stl / arm_front.svg) -- print one mirrored
  2x rear arm (arm_rear.stl / arm_rear.svg) -- print one mirrored
  1x main plate (plate_main.stl / plate_main.svg)
  1x mid plate (plate_mid.stl / plate_mid.svg)
  1x top plate (plate_top.stl / plate_top.svg)

STLs are ready to print; SVGs are true-scale (mm) outlines
for CNC/laser cutting. All parts are flat in print/cut
orientation. Bolt holes and cutouts come from the real
Source One V6 outlines; hardware (standoffs, screws, stack,
motors) is the stock kit's.

The arm-clamp bolt holes in the main and mid plates are
RE-CUT to match this candidate's actual arm sweep and plate
scale (stock clamp holes that would overlap were removed), so
the arms bolt straight on. Note the simulated frame carried
the scaled stock hole pattern instead -- the difference is a
few M3 holes' worth of material.
